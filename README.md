# Social-Network-API

## Purpose

To provide the user an API for their social network that uses a NoSQL database so that their website can handle large amounts of unstructured data.

---

## Technology Used

• Node
• Express.js
• Javascript
• Mongoose
• MongoDB
• Insomnia
